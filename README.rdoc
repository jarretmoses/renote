== README
Authors: Jarret Moses, Fabio Kenji, Nathan Kratchman

Rails app that allows users to bookmark important sections of a video and take personal notes at the same time

AJAX used for asynchronous database queries and for a enhanced user experience
Video.js /videojs-markers plugins used for customized video integration for necessary "bookmarking" functionality
jQuery and Javascript used for optimal UI functionality
<tt>rake doc:app</tt>.
